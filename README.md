# SMARTICUS

A Python-based interactive BuzzFeed-style quiz game built as a class project 
at Delhi Public School, R.K.Puram, New Delhi.

Built by: Sanaa Sharda 

## What It Does
SMARTICUS is a command-line quiz game with 3 personality quizzes that 
recommend results based on user responses using a point-based scoring system.

- Quiz 1: Answer food questions to get a movie recommendation
- Quiz 2: Answer personality questions to find out which dorm food you are
- Quiz 3: Answer preference questions to get a song recommendation

## How It Works
Each quiz asks the user a series of multiple choice questions. Points are 
assigned per answer and totaled at the end to determine a result. 
The game uses loops and conditional logic to handle input validation, 
allow users to quit mid-quiz, and restart on invalid input.

## Features
- Input validation with error handling
- Quit functionality at any point (press Q)
- Point-based scoring system across multiple questions
- 3 independent quizzes with unique themes and results
